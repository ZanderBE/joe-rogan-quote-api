The Joe Rogan API Experience

For a single quote return:

https://joeroganapi.glitch.me/quotes

For a multiple quote return:

Replace num with the amount of quotes you want returned. (max 50)

https://joeroganapi.glitch.me/quotes?count=num

Example return:

[
  {
    quote: "There is a direct correlation between positive energy and positive results.",
    author: "Joe Rogan"
  }
]
